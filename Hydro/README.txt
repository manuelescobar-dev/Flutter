# Hydro
Next-gen hydroponics app.

Prototype for a new hydroponics app using Flutter. The app will be able to control a hydroponics system using a Raspberry Pi and Arduino.

Features:
- Manage different plants and their needs
- Manage different hydroponics systems
- Manage different sensors and their data
- Manage different actuators and their data
- Manage different users and their permissions
- Innovative design
